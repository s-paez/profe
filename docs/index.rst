PROFE Documentation
===================

**PROFE** (*Pipeline de Reducción de OPTICAM para Fotometría de Exoplanetas*) is a
Python-based pipeline to automate preprocessing and postprocessing of data acquired
with the OPTICAM instrument on the 2.1 m telescope at OAN-SPM, implementing the data
reduction method proposed by `Paez et al. (2026) <https://doi.org/10.1093/rasti/rzag021>`_.
